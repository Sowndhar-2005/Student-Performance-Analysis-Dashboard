# Data Science Project Plan

## Stack
- Backend: Python + Flask + Pandas + Scikit-learn + Matplotlib/Seaborn
- Frontend: Vanilla HTML/CSS/JS with Chart.js for interactive charts
- Dataset: Synthetic student performance data (generated on startup)

## Features
1. Dashboard with KPI cards
2. Data Explorer (grid + filters)
3. Visualizations (scatter, histogram, correlation heatmap)
4. ML Model (predict pass/fail with logistic regression)
5. Insights & Report tab
